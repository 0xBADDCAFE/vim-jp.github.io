# Vim Magazine 2 月号

## 話題

- [Vim Advent Calendar 2012](http://atnd.org/events/33746)

  Vim Advent Calendar 2012 まだやってます。


## 今月の新機能

- 7.3.826 : `:version` の出力が見やすくなりました。
- 7.3.834 : if_ruby が Ruby 2.0 に対応しました。
- 7.3.836 : Cygwin 版でクリップボードが使えるようになりました。
